DINNERPLATE

I had previously read somewhere that orange mugs make drinks taste batter, which inspired me to utilize an inviting orange for the visual identity of Dinnerplate. By using this orange, it could possibly appetize the user while choosing where to eat. A targeted audience are young adults, which explains the simple and linear format for the app.
